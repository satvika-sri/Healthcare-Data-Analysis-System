# Healthcare-Data-Analysis-System
Developed a healthcare data analytics solution using Apache NiFi and Kafka for real-time data ingestion and streaming, Apache Hive and Spark for large-scale data processing and analysis, and Power BI for interactive dashboards.
## Streaming and Automation
1.Implemented Spark Structured Streaming for ingesting patient monitoring data.
2.Simulated real-time ingestion using Kafka and nifi 
## PowerBI reporting
PowerBI dashboards were created to visualize :
1. Disease prevalence by gender
2. count of product name and sum of max price by company name
3. sum of people in 2022 by Disease name and gender
4. count of patient I'd and sum of total prescriptions
5. sum of treatment count by city and Disease name
